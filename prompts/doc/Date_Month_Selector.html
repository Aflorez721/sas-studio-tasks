Date Month Selector Task

The date month selector task displays an Options tab containing a combo box. The choices in the combobox are:
  • Choose a month - a month combobox and year input field are shown
  • Current month
  • Previous month
  • Next month
  • Current month of previous year
  • Current month of next year
  • N months ago - an input field prompts for an integer
  • N months from now - an input field prompts for an integer
When a selection made and values are specified in any dependent controls the following macro variables are created and set as follows:
  • month - the first day of the month selected or a function that calculates the first day of the month of the selection, formatted as DDMMMYYYY
  • month_end - the last day of the month selected or a function that calculates the last day of the month of the selection, formatted as DDMMMYYYY
  • month_rel - a code that represents the selection made
  • month_label - the common name of the selection

An example of the code generated by this task when "Current month" is selected: 

%global month;
%global month_end;
%global month_rel;
%global month_label;
%let month = %sysfunc(intnx(Month, %sysfunc(date()), 0, b), DATE9.);
%let month_end = %sysfunc(intnx(Month, %sysfunc(date()), 0, e), DATE9.);
%let month_label = Current Month;
%let month_rel = M0M;
%put _GLOBAL_;

The macro variable values set by this example code if this month is March of 2017 are: 

GLOBAL MONTH 01MAR2017
GLOBAL MONTH_END 31MAR2017
GLOBAL MONTH_LABEL Current Month
GLOBAL MONTH_REL M0M
