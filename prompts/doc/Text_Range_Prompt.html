Text Range Prompt Task

The text range prompt task displays an Options tab containing From and To input fields. When values are specified macro variables are created as follows:
  • textrange_min - set to the From value specified
  • textrange_max - set to the To value specified

An example of the code generated by this task if values in the From and To input fields are a and z. 

%global textrange_min;
%global textrange_max;
%let textrange_min=a;
%let textrange_max=z;
