Multiple Text Task

The muliple text prompt task displays an Options tab containing a SAS Studio multientry field. When values are specified a set of macro variables are created as follows:
  • textselector_count - set to the number of values specified in the multientry field
  • textselector0 - set to the number of values specified in the multientry field
  • textselector - set to the first value specified in the multientry field
  • textselectorN where N is 1...N - set to the values specified in the multientry field

An example of the code generated by this task if values in the multientry field are "one", "two", and "three" 

%global textselector_count;
%let textselector_count=3;
%global textselector0;
%let textselector0=3;
%global textselector;
%let textselector=one;
%global textselector1;
%let textselector1=one;
%global textselector2;
%let textselector2=two;
%global textselector3;
%let textselector3=three;
