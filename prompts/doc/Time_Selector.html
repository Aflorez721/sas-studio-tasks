Time Selector Task

The time selector task displays an Options tab containing a combo box. The choices in the combobox are:
  • Specify time - an input field prompting for a time appears
  • Current hour
  • Previous hour
  • Next hour
  • Current minute
  • Next minute
  • Previous minute
  • N minutes ago - an input field prompts for an integer
  • N minutes from now - an input field prompts for an integer
  • N hours ago - an input field prompts for an integer
  • N hours from now - an input field prompts for an integer
When a selection made and values are specified in any dependent controls the following macro variables are created and set as follows:
  • time - the time specified or a function that calculates the value of the selection, formatted as HH:MM:SS
  • time_rel - a code that represents the selection made
  • time_label - the common name of the selection

An example of the code generated by this task when "Current time" is selected: 

%global time;
%global time_rel;
%global time_label;
%let time = %sysfunc(time(), time8.0);
%let time_label = Current time;
%let time_rel = t0m;

The macro variable values set by this example code are as follows if it is 11:27:27 AM. 

 GLOBAL TIME 11:27:27
 GLOBAL TIME_LABEL Current time
 GLOBAL TIME_REL t0m
