Hyperlink Prompt Task

The hyperlink prompt task displays an Options tab containing input fields for "Link text" and "Link address(URL)". When values are specified the following macro variables are created and set as follows:
  • Folder_Server - set to the value "Local"
  • Folder - the complete path of the folder selected

An example of the code generated by this task is: 

%global Hyperlink;
%let Hyperlink_Path="Local";
%global Hyperlink;
%let Hyperlink=/r/ge.unx.sas.com/vol/vol810/u81/sasxxx;
