Folder Selector Task

The folder selector task displays an Options tab containing a folder selector. When a selection made he following macro variables are created and set as follows:
  • Folder_Server - set to the value "Local"
  • Folder - the complete path of the folder selected

An example of the code generated by this task is: 

%global Folder_Server;
%let Folder_Server="Local";
%global Folder;
%let Folder=/r/ge.unx.sas.com/vol/vol810/u81/sasxxx;
