Number Range Prompt Task

The number range prompt task displays an Options tab containing From and To input fields. When values are specified macro variables are created as follows:
  • NumRange_Min - set to the From value specified
  • NumRange_Max - set to the To value specified

An example of the code generated by this task if values in the From and To input fields are 1 and 10,000. 

%global NumRange_Min;
%let NumRange_Min=1.0;
%global NumRange_Max;
%let NumRange_Max=10000.0;
