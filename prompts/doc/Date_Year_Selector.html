Date Year Selector Task

The date year selector task displays an Options tab containing a combo box. The choices in the combobox are:
  • Select a year - a year input field prompt for a year
  • Current year
  • Previous year
  • Next year
  • N years ago - an input field prompts for an integer
  • N year from now - an input field prompts for an integer
When a selection made and values are specified in any dependent controls the following macro variables are created and set as follows:
  • year- the first day of the year selected or a function that calculates the first day of the year of the selection, formatted as DDMMMYYYY
  • year_end - the first last day of the year selected or a function that calculates the last day of the year of the selection, formatted as DDMMMYYYY
  • year_rel - a code that represents the selection made
  • year_label - the common name of the selection

An example of the code generated by this task when "Next year" is selected: 

%global year;
%global year_end;
%global year_rel;
%global year_label;
%let year = %sysfunc(intnx(Year, %sysfunc(date()), 1, b), DATE9.);
%let year_end = %sysfunc(intnx(Year, %sysfunc(date()), 1, e), DATE9.);
%let year_label = Next Year;
%let year_rel = Y1Y;
%put _GLOBAL_;

If this year is 2017 the macro variable values set by this example code are: 

 GLOBAL YEAR 01JAN2018
 GLOBAL YEAR_END 31DEC2018
 GLOBAL YEAR_LABEL Next Year
 GLOBAL YEAR_REL Y1Y
